This is an alternative GGPO command line client written in python. I have only tested this on Linux but it might work on other platforms. You'll still need the ggpofba.exe from the original GGPO, which will be launched through wine.

Installation

Extract the original GGPO in /opt/ggpo, and add ggpo.py and ggpofba.sh in the same folder. Edit ggpo.py to set your username and password.

Usage

$ /opt/ggpo/ggpo.py
ggpo> /help
-!- available commands:
-!- /challenge    send a challenge request to 
-!- /cancel       cancel an ongoing challenge request to 
-!- /accept       accept a challenge request initiated by 
-!- /decline      decline a challenge request initiated by 
-!- /watch        watch the game that  is currently playing
-!- /join      join the chat/game room 
-!- /list       list all available channels or chat/game rooms
-!- /users      list all users in the current channel
-!- /intro      view the channel welcome text
-!- /away       set away status (you can't be challenged)
-!- /back       set available status (you can be challenged)
-!- /clear      clear the screen
-!- /quit       quit ggpo
